1. **Upload PHP API**

   - To: `public_html/api/`

2. **Upload Frontend**
   - From: `dist/` folder
   - To: `public_html/`

3. **Upload Images**
   - From: `downloaded-images/` folder
   - To: `public_html/images/trips/`
